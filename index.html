Amazon CloudFront

Amazon CloudFront adalah layanan Content Delivery Network (CDN) yang disediakan oleh Amazon Web Services (AWS). CloudFront berfungsi untuk mempercepat pengiriman konten web kepada pengguna dengan menggunakan jaringan server global yang disebut edge locations.

Amazon Elastic Compute Cloud (Amazon EC2)

Amazon Elastic Compute Cloud (Amazon EC2) adalah layanan dari Amazon Web Services (AWS) yang menyediakan kapasitas komputasi yang dapat diskalakan di cloud. EC2 memungkinkan pengguna untuk menjalankan virtual machines (VMs) di infrastruktur AWS dengan cara yang fleksibel dan terukur.

Amazon S3 (Simple Storage Service)

Amazon S3 (Simple Storage Service) adalah layanan penyimpanan cloud yang ditawarkan oleh Amazon Web Services (AWS). Layanan ini dirancang untuk menyimpan dan mengelola data dalam jumlah besar secara mudah dan aman.
